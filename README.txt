npm install
npm start -команда которая запустит dev сервер, необходимо для авоматической компиляции scss/ES6 файлов
Перейти в браузере по адресу http://localhost:8080
Для остановки дев сервера в консоле(терминале) необходимо нажать два раза CTRL+C
___________________________________________________________________________________________________________

/assets - папка для картинок, шрифтов и тд.
/index, secondPage - страницы + сопуствующие файлы
/style -  папка для общих стилей для всех страницы
main.js - точка входа для всех страниц, подключаем модули и библиотеки которые все для одного проекта, так же общие стили сюда подключаем и тд.
